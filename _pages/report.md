# 基于PIC数据库的儿科ICU死亡风险预测

## 1. 研究背景
在重症监护病房中，及时识别高危患儿对改善预后至关重要。  
本作业使用了公开的 **PIC（Paediatric Intensive Care）** 数据库，该数据库包含了国内一家大型儿童医院儿科ICU脱敏后的真实临床数据。  
我从中提取了患儿入ICU后 **24小时内** 的临床指标，尝试用机器学习算法来预测院内死亡风险。

## 2. 数据与方法
- **样本量**：原始数据共 **13258** 例，特征数 **1666** 个。
- **数据预处理**：缺失值使用了中位数（数值型）和众数（类别型）填充，并将所有类别变量进行编码。
- **数据集划分**：按 7:3 随机分层划分为训练集和测试集。
- **模型选择**：考虑到可解释性和运算效率，这里选用了 **随机森林 (Random Forest)**。参数设置为：树的数量=100，随机种子=42。

## 4. 讨论与结论
- 这次实验表明，基于常规生理指标和实验室检查数据，随机森林可以较为准确地预测儿科ICU患儿的院内死亡风险（AUC ≈ 0.841）。
- 模型给出的重要特征可作为临床决策的参考，但实际应用仍需结合医生判断。
- 后续可以尝试加入更多时序信息、使用深度学习等方法进一步提升性能。
